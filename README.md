🇬🇧 UK Job Market Dashboard
Tool: Tableau Public  |  Dataset: 1.6M job postings (Kaggle)
An interactive dashboard analysing the UK job market across three dimensions — top hiring companies, work type distribution, and experience level requirements.
Key insights:
•	DTE Energy and Midea Group lead UK job postings in this dataset
•	Work type is evenly distributed across Contract, Full-Time, Intern, Part-Time and Temporary roles
•	Experience brackets from 0–15 years are well represented, with mid-range experience (1–12 years) most in demand
Skills demonstrated: Data filtering, aggregation, COUNT measures, dashboard composition, Tableau Public publishing

How I Work
Every project in this repo follows the same approach:
1.	Define the question — what story is this data supposed to tell?
2.	Explore and clean — understand the shape, fix the mess
3.	Analyse — aggregations, trends, distributions, outliers
4.	Visualise — build something a non-technical person can read in 30 seconds
5.	Document — write up the findings clearly
